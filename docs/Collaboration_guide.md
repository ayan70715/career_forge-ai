# 📌 CareerForge-AI Collaboration Guide

Repository: https://github.com/Techlead-ANKAN/CareerForge-AI.git

This file explains how both developers should work on this project.

------------------------------------------------------------
🚀 FIRST TIME SETUP (Only Once)
------------------------------------------------------------

1. Clone the repository:

git clone https://github.com/Techlead-ANKAN/CareerForge-AI.git

2. Go inside the project folder:

cd CareerForge-AI


------------------------------------------------------------
🛠 BEFORE STARTING WORK (Every Time)
------------------------------------------------------------

Always pull the latest changes before working:

git pull origin master


------------------------------------------------------------
💻 AFTER MAKING CHANGES
------------------------------------------------------------

1. Add changes:

git add .

2. Commit changes:

git commit -m "Write what you changed"

3. Push changes:

git push origin master


------------------------------------------------------------
⚠ IMPORTANT RULES
------------------------------------------------------------

• Always run "git pull" before starting work.
• Write clear commit messages.
• Do NOT delete important project files.
• Inform teammate before making major changes.


------------------------------------------------------------
🔥 IF PUSH IS REJECTED
------------------------------------------------------------

If you see an error like:

"rejected because remote contains work that you do not have locally"

Run:

git pull origin master

Resolve conflicts if any, then:

git push origin master


------------------------------------------------------------
📌 SIMPLE WORKFLOW SUMMARY

Clone (only once)
Pull (before working)
Add → Commit → Push (after working)
